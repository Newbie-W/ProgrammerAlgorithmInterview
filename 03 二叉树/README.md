### 二叉树
-----
#### 一、常用知识及常见笔试题
1. 二叉树基础知识
2. 如何把一个有序整数数组放到二叉树中
> **题目**：把一个有序整数数组放到二叉树中。<br>
3. 如何从顶部开始逐层打印二叉树结点数据
> **题目**：层序遍历，按层的顺序，从上至下，从左至右，逐层打印出二叉树的结点数据。<br>
> **引申**：用空间复杂度为O(1)的算法来实现层序遍历<br>
4. 如何求一棵二叉树的最大子树和
> **题目**：给定一棵二叉树，它的每个结点都是正整数或负整数，如何找到一棵子树，使得它所有结点的和最大？<br>
5. 如何判断两棵二叉树是否相等
> **题目**：两棵二叉树相等，是指这两棵二叉树有着相同的结构，并且在相同位置上的结点有相同的值。如何判断两棵二叉树是否相等？<br>
