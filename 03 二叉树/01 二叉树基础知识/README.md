### 一、二叉树基础知识
1. **定义相关**：二叉树（Binary Tree）,又称 二分树、二元树、对分树等。它是n(n>=0)个有限元素的集合，该集合或者为空、或者由一个称为根（root）的元素及两个不相交的、分别称为左子树和右子树的二叉树组成。当集合为空时，称该二叉树为空二叉树。<br>
在二叉树中，一个元素也称作一个结点。二叉树的递归定义：二叉树或者是一棵空树，或者是一棵由一个根节点和两棵互不相交的分别称作根结点的左子树和右子树所组成的非空树，左子树和右子树又同样都是一棵二叉树。<br>

2. 一些常见的二叉树的**基本概念**：（加:exclamation:的，为个人记忆不深刻的）<br>
（1）:exclamation:**结点的度**：结点所拥有的**子树的个数**称为该结点的度。<br>
（2）**叶结点**：**度为0**的结点称为叶结点（又，终端结点）<br>
（3）**分支结点**：**度不为0**的结点。（又，非终端结点）一棵树的结点除叶结点外，其余的均为分支结点。<br>
（4）**左孩子、右孩子、双亲**<br>
（5）:exclamation:**路径、路径长度**：如果一棵树的一串结点$n_1, n_2, ..., n_k$，有如下关系：结点$n_i$是$n_{i+1}$的父节点（$1\leq i<k$），就把$n_1, n_2, ..., n_k$称为一条由$n_1$到$n_k$的路径，这条路径的长度为```k-1```<br>
（6）**祖先、子孙**：在树中，如果由一条路径从结点M到结点N，那么M就称为N的祖先，N称为M的子孙。<br>
（7）:exclamation:**结点的层数**：规定树的根结点的层数为1，其余结点的层数等于它的双亲结点的层数+1。（简言之，**根结点层数=1，其他结点层数=其父结点层数+1**）<br>
（8）:exclamation:**树的深（高）度**：树中所有**结点的最大层数**。<br>
（9）:exclamation:**树的度**：树中各结点度的最大值。叶子结点的度为0。<br>
（10）**满二叉树**：在一棵二叉树中，如果所有分支结点都存在左子树和右子树，并且所有叶子结点都在同一层上，这样的二叉树称为满二叉树。<br>
（11）**完全二叉树**：一棵深度为k的、有n个结点的二叉树，对树种的结点按从上至下、从左至右的顺序进行编号，如果编号i($1 \leq i \leq n$)的结点与满二叉树中编号为i的结点在二叉树中的位置相同，那么这棵二叉树称为完全二叉树。<br>
完全二叉树的**特点**是，叶子结点只能出现在最下层和次下层，且最下层的叶子结点集中在树的左部。<br>
满二叉树一定是完全二叉树，完全二叉树不一定是满二叉树。<br>

3. 二叉树的**基本性质**:
- （1）一棵非空二叉树的第i层上最多有$2^{i-1}$个结点（i \geq 1）。
- （2）一棵深度为k的二叉树中，最多具有$2^k - 1$个结点，最少有k个结点。
- （3）对于一棵非空二叉树，度为0的结点（叶子结点），总是比度为2的结点多一个，即如果叶子结点个数为$n_0$，度为2的结点数为$n_2$， 则有$n_0 = n_2 + 1$
> 证明：<br>
> 用$n_0$表示度为0的结点（叶子结点）的结点总数，$n_1$表示度为1的结点总数，$n_2$为度为2的结点总数；n表示整个二叉树中的结点总数。<br>
> 一方面，结点总数n = 度为0的结点数$n_0$ + 度为1的结点数$n_1$ + 度为2的结点数$n_2$。（$ n = n_0 + n_1 + n_2 $）<br>
> 另一方面，**结点总数=所有结点的度数+1**。度为0的结点没有孩子，度为1的结点有1个孩子，度为2的结点有两个孩子结点。故，二叉树中的孩子结点数为$ n_1+ 2 \ast n_2 $。再加上没有父结点的根结点，得$ n = n_1 + 2 \ast n_2 + 1 $<br>
> 由两个等式即可得到$n_0 = n_2 + 1$<br>
- （4）具有n个结点的完全二叉树的深度为 $ \lfloor \log_2 n \rfloor + 1$
> 证明：<br>
> 根据性质2，深度为k的二叉树最多只有$2^{k-1}$个结点，其总结点数n满足范围$2^{k-1} - 1 < n \leq 2^k - 1$或$2^{k-1} \leq n < 2^k$，三边同取对数，得$k-1 \leq \log_2 n < k$ （$ \log_2 n < k \leq log_2 n + 1 $）， k为整数，故$ k = \lfloor \log_2 n \rfloor + 1 $ <br>
- （5）对于具有n个结点得完全二叉树，如果按照从上至下、从左至右的顺序对二叉树中得所有结点从1开始顺序编号，那么对于任意的序号i的结点有
> 1. 如果 $ i > 1 $，则序号为i的结点的双亲结点的序号为 i/2 （/为整除符号）； <br>
> 2. 如果 $ 2 \ast i \leq n $，那么序号为i的结点的左孩子结点的序号为$ 2 \ast i $；如果 $ 2 \ast i > n $，那么序号为i的结点无左孩子；<br>
> 3. 如果 $ 2 \ast i + 1 \leq n $，那么序号为i的结点的右孩子结点的序号为$ 2 \ast i + 1 $；如果 $ 2 \ast i + 1 > n $，那么序号为i的结点无右孩子；<br>
> <br>
> 假设二叉树的根结点从1开始编号，则如果一个结点编号为i，其具有左右孩子结点，那么其左孩子结点编号为$2 \ast i$，右孩子结点编号为$ 2 \ast i + 1 $。（书中提到的是以0为根结点的编号，左右孩子结点的取值相对会改变，可自己画图举例子判断值）<br>

4. **例题**：
（1）一棵完全二叉树上有1001个结点，其中叶子结点的个数是多少？<br>
**分析**：根据二叉树公式，$ n = n_0 + n_1 + n_2 = n_0 + n_1 + (n_0 - 1) = 2 \ast n_0 + n_1 - 1 $。而在**完全二叉树中，$n_1$只能取0或1**。从而可得 $ n_0 = 501 $ <br>
（2）如果根的层次为1，具有61个结点的完全二叉树的高度为？<br>
**分析**：根据二叉树的性质，**具有n个结点的完全二叉树的深度为$ \lfloor \log_2 n \rfloor + 1$**。因此，具有61个结点的完全二叉树的高度为$ \lfloor \log_2 61 \rfloor + 1 = 5 + 1 $。最终答案为6。<br>
（3）在具有100个结点的树中，其边的个数为多少？<br>
**分析**：在一棵树中，除了根结点之外，每个结点都有一条边指向它，因此总边数应该为 100 - 1 = 99 条。<br>


#### 二、单链表的定义
二叉树有顺序存储和链式存储，本书本章涉及的算法均采用链式存储结构，具体实现如下（即BinaryTree.h）：
```C++
/*树结点数据结构*/
typedef struct BiTNode {
	ElemType data;		// 数据域，本章算法假设ElemType为int
	struct BiTNode *lchild, *rchild;		// 左右孩子指针
};
```
