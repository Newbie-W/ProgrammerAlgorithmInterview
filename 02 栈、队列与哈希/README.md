### 栈、队列与哈希
-----
#### 一、
- 栈与队列是在程序设计中广泛使用的两种重要的线性数据结构，都是在一个特定范围的存储单元中存储的数据，这些数据都可以重新被取出使用。与线性表相比，它们的插入和删除操作受到更多的约束和限定，故又称为限定性的线性表结构。
- 1. 栈：  LIFO（Last In First Out， 后进先出）
- 2. 队列：FIFO（First In First Out， 先进先出）

- 需要注意的是，有时在数据结构中还有可能出现按照大小排队或按照一定条件排队的数据队列，这时的队列属于特殊队列，就不一定按照“先进先出”的原则读取数据了。<br>

#### 二、常见笔试题
1. 如何实现栈
> **题目**：实现一个栈的数据结构，使其具有以下方法：压栈、弹栈、取栈顶元素、判断栈是否为空以及获取栈中元素个数。<br>
2. 如何实现队列
> **题目**：实现一个队列的数据结构， 使其具有【入队列】、【出队列】、【查看队列首尾元素】以及【查看队列大小】等功能<br>
3. 如何翻转栈的所有元素
> **题目**：翻转（也叫颠倒）栈的所有元素，例如输入栈{1, 2, 3, 4, 5}，其中，1处在栈顶，翻转之后的栈为{5, 4, 3, 2, 1}，其中，5在栈顶。<br>
> **引申**：如何给栈排序<br>
4. 如何根据入栈序列判断可能的出栈序列
> **题目**：输入两个整数序列，其中一个序列表示栈的push（入栈）顺序，判断另一个序列有没有可能为对应的pop（出栈）顺序。<br>
